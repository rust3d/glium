Specifies which comparaison the GPU will do to determine whether a sample passes the stencil test.

Variants

AlwaysPass

The stencil test always passes.

AlwaysFail

The stencil test always fails.

IfLess

Applies the mask as a bitfield to both the value currently in the stencil buffer and the reference value (stencil_reference_value_clockwise or stencil_reference_value_counter_clockwise), then compares them.
